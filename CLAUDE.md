# 通用内容生产 agent

> 目标：把内容生产引擎通用化，任意创作者可用（区别于 `~/新媒体agent` 只 focus 迪迪本人）。
> 设计讨论见 `设计讨论-session0.md`。本项目尚在设计阶段。

## 🔝 最高铁律（凌驾一切，冲突时以此为准）

1. **第一性原理思考与复盘**：从根本事实往上推，不靠类比、惯例、"别人都这么做"、现成结论。复盘归因必须问到根因（指标到底在测什么、机制上为什么），别停在表层模式匹配；数据与套路冲突时信数据、拆机制。
2. **省 token 优先**：结果先行、工具先行；表格/清单 > 大段文字；不复述、不铺垫、不输出没被要的东西。仅"写文案正文"例外（保人味）。
