# 违禁词库 + 白名单（预审召回网·融自 traffic-compass）

> 用法：预审时**先拿这个词库扫文案/标题/tag（召回网，见词先标）→ AI 读语境判真假 → 白名单防误杀 → 命中走"最小改动"只换那几个词**。
> 词库负责"别漏"，白名单负责"别误伤"。判断总原则：是不是在**宣传商品/服务、承诺效果、导流站外**——是才算违规，主观表达/序数/科普引用不算。
> 本表是跨平台通用词法（广告法等），平台特有词表见各平台模块，账号定位红线见 `profile.redlines`。

---

## 一、违禁/风险词（分类召回网）

**🔴 广告法极限词（绝对化，必改）**
最 / 第一 / 唯一 / 首个 / 顶级 / 顶尖 / 极致 / 最佳 / 最好 / 最强 / 最高级 / 国家级 / 世界级 / 全球级 / 100% / 满分 / 史上 / 绝无仅有 / 独一无二 / 一流 / 领先 / 王牌 / 冠军 / 永久 / 万能
→ 改成可验证具体说法（"最好用的"→"挺好用的"；"第一"→去掉或"名列前面"）。

**🔴 夸大功效 / 收益承诺（必改）**
根治 / 治愈 / 彻底解决 / 包过 / 包会 / 包录取 / 稳赚 / 躺赚 / 月入过万 / 三天见效 / 一夜暴富 / 财富自由 / 轻松月入
→ 换实证调性（"我真做出来了/我的实测"），别承诺结果。

**🔴 导流引流（限流高发·必改）**
微信 / 微信号 / 加V / 加威 / vx / 私我 / 私信我发你 / 主页链接 / 公众号 / 二维码 / 滴滴我 / 扣我 + 明示站外
→ 改"评论区扣【口令】我发你"（按 `profile.monetization` 的导流方式）；画面/口播别出现微信号、二维码。

**🔴 封建迷信**
转运 / 开光 / 风水 / 算命 / 改命 / 招财符 / 本命佛 / 化太岁 / 玄学改运

**🟡 焦虑营销（读语境，过度才改）**
被取代 / 失业潮 / 不学就被淘汰 / 时代抛弃你 / 最后的机会 / 再不学就晚了
→ 适度点痛点可以，**贩卖焦虑+制造恐慌**才标黄；改成给方法、给安心的正向框。

**🔴 资质门槛（无资质别碰）**
医疗（诊断/荐药/疗效）、金融（荐股/收益承诺/抄底）、法律（法律意见）、教育（升学/录取承诺）。详见 `通用限流诱因.md`。

## 二、白名单（防误杀·这些语境不算违规）

- **"最/第一"用在非商业主观表达**：`我最喜欢` `对我最有用` `第一次用` `第一集` → 不是给商品/服务背书，不算极限词。序数（第一次/第一集）≠ 极限词"第一/唯一"。
- **引用/科普里提到某词** ≠ 宣传该行为：讲"别人算命被骗"不算封建迷信营销。
- **口语强调**：`真的绝了` `太强了`（形容感受，非商品广告词）→ 保留，别改成 AI 腔。
- **判断闸门**：一个词要标红，必须同时满足①在宣传商品/服务/承诺效果 或 导流站外 ②不是主观感受/序数/科普引用。**只见词不看语境=误杀**。

> 误杀比漏检更伤——把作者口语腔改没了就违背人味儿（见 `共享机制/人味儿写作.md`）。拿不准时标🟡"可能有风险"提示人工看，别直接判🔴。
