# 通用限流诱因（跨平台,所有平台叠加）

> 这些是各短视频/图文平台共性的"会限流/仅自己可见"诱因,预审第一优先逐条查。

**🔴 站外导流 / 联系方式**
- 文案贴提示词全文/口令明文/微信号/手机号/外部链接/二维码 → 改成"评论区扣【XX】我发你"。
- 明示引导站外("加我微信""去主页链接") → 含蓄化,按 `profile.monetization` 的导流方式。

**🔴 搬运 / 非原创**
- 原样搬别人截图/视频/文案 → 必须有自己的创作增量 + 标注来源。

**🟡 违规/夸大词（平台法规通用）**
- 绝对化:最/第一/唯一/100%(广告法)。
- 医疗功效、投资收益承诺、夸大效果。
- 标题党但正文无干货。

**🟡 营销违规**
- 未授权的抽奖/送礼/打折促销。变现用软方式(评论区口令导私域),别站内硬促销。

**🔴 AI 生成内容标注（多平台已强制·最容易漏）**
- 内容含 AI 生成/合成的图、视频、音频、文案 → 发布时需**显著声明"AI 生成"**(各平台发布页有对应勾选/打标入口,实现见平台模块)。不标风险:被平台补标 / 降权 / 限流。
- 对 AI 类账号(内容大量 AI 生成)是硬红线;口播主动说"这条我用 AI 做的"反而强化人设。

**🔴 资质门槛（无资质别碰·法规通用）**
- 医疗(诊断/荐药/疗效)、金融(荐股/收益承诺/抄底)、法律(法律意见)、教育(升学/录取承诺/包过)。滑进这些领域给建议要立刻收住。

> 平台特定的(如抖音"标题英文=搬运嫌疑"、小红书违禁词表、AI 声明的具体勾选入口)见各平台模块;完整词表见 `违禁词库与白名单.md`。
