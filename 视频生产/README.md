# 视频生产（基础库 · 提供给用户）

> 文案定稿后的出片环节。本目录是**基础模板库**,任何创作者都能挑。
> 视觉风格不写死迪迪——**配色/封面/幻灯片风格由 `profile.visual_style` 决定**,模板是共用骨架。

## 基础库内容（模板库/）
- `幻灯片风格.md` —— SL-01 亮底编辑撞色 / SL-02 涂鸦 / SL-04 表格图等风格骨架(配色按 profile)。
- `画面与动效.md` —— 通用动效手法(数字对撞/逐条弹出/节点点亮)。
- `编排方案模板.md` —— 录前逐段规划表(口播要点→展示方式→素材分工)。
- `封面_SwissIKB模板.html` —— 封面骨架(Swiss 版式),配色/文案/照片按 profile 替换。
- `博主拆解.md` —— 拆解卡视觉模板。

## 怎么用 profile 驱动
```
读 profile.visual_style:
  palette         → 幻灯片/封面的配色(迪迪宝蓝撞红;职场琳暖绿撞橙)
  slide_template  → 选哪个 SL 风格
  cover/cover_template + must_face → 封面怎么出、要不要真人脸
```
同一套模板骨架,换 profile 换配色和封面规则——和选题/复盘一样,机制共用、值来自档案。

## 出片工具链（本基础库只放模板,渲染引擎按下面选）
- **封面/图卡**:让 agent 用 PIL 按 `visual_style.palette` + 本目录模板直接出 PNG(参考 `封面_SwissIKB模板.html` 的版式)。
- **成片渲染**:推荐装 **hyperframes** skill(HTML→视频,带 TTS/字幕/转场,门槛低);或自备 remotion 管线。两者都没有 → 出"编排方案+素材包",用户用剪映等手剪。
> 渲染类工具先预览定稿再导出,别上来就跑全片渲染。

## 流水线位置
`文案(定稿+发布包)` → **视频生产(本环节:按 visual_style 出封面/幻灯片/编排)** → 成片发布 → `复盘`。
见 `工作流.md`(全链)。
