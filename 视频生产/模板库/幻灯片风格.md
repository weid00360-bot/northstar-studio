### [SL-01] 亮底编辑撞色 — ✅ 已沉淀（迪迪主用）
- 来源：小红书模板 ref2(Project Proposal) + ref3(South Korea)
- 适用：干货/教程/数据，要权威又清晰
- 配色：奶油底 + 宝蓝/红撞色
- 字体：超大粗黑体，关键词换主色

- 实现：PIL 已验证；Remotion 用 div + 思源黑；手绘元素 SVG path
- 样张：`迪迪幻灯片_新风格.png`

### [SL-02] 活泼涂鸦（SL-01 变体）— ✅ 已沉淀
- 来源：ref4(Green Jobs 涂鸦小人)，掺进 SL-01
- 适用：轻松段落、build-in-public、情绪点

### [SL-04] 表格图（SL-01 变体）— ✅ 已沉淀
- 来源：迪迪 agent 真实框架（`套路库.md` / `skills/`）做成网格表
